## 4. OCR 파싱 시 주의사항

1. **사진 회전**: 모든 사진이 90도 회전되어 있음 → 이미지 회전 전처리 필요
2. **다양한 날짜 형식**: 매장마다 8가지 다른 형식 → 다중 패턴 매칭 필요
3. **중국어/한국어 병기**: 아시안 마트 영수증에 혼합 언어 → 다국어 OCR 고려
4. **무게 기반 가격**: `0.454kg @ $6.90/kg` 형식 → 수량×단가 파싱
5. **Rounding**: 호주 현금 결제 시 5센트 반올림 → 소계와 총액이 다를 수 있음
6. **할인 표시**: 다양한 형식 (15% Off, Total Savings, You have saved)
7. **매장명 위치**: 대부분 상단이지만, 일부는 잘려 있을 수 있음
8. **TOTAL이 가장 신뢰도 높음**: 개별 품목보다 총액이 크고 선명하게 인쇄됨
